# BluetoothScanner

### 📝 Description
-	CoreBluetooth를 활용한 블루투스 통신 앱
***

### 🔎 Key Features
-	검색된 기기 중 원하는 기기 연결 시도 가능
-	연결 성공시 service, characteristic 목록 조회 가능
***

### ❕ Special Notes
-	테스트 기기가 없어 원하는 service id - chracteristic id 연결 테스트가 불가능하여 일시 중단 상태
-	기기 확보할 경우 notify, read, write 기능 추가 예정
***


### ⌨️ Language
- Swift
***

### 🛠️ Library
- SwiftUI
- CoreBluetooth
***
